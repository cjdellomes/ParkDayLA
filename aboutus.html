About ParkDayLA: Summary!

ParkDayLA is a website designed to provide information about parking areas when an address is searched. Currently only the Santa Monica area is supported.

By inputting the desired address, a user can find whether or not the street is metered, find local parking areas, and find spots avaiable at nearby parking structures.

A map and street view is given so that the user can check firsthand the layout of the street and parking.
/
A user can now see whether or not parking is available at their desired location and in a 3 miles radius.

Google Maps API and Santa Monica Parking API
